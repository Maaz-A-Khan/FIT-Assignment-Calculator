Calculator App

Simple Calculator

Project Overview
This project is a simple calculator program written in C that can perform basic arithmetic operations: addition, subtraction, multiplication, and division.

How to Run the Calculator
1. Clone the repository:
git clone <repository-url>

2. Navigate to the directory:
cd calculator

3. Compile the program:
gcc calculator.c -o calculator

4. Run the program:
./calculator
